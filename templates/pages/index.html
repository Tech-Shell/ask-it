Features

Filtered Inputs

Every input is filtered first, removing the unnecessary and inappropriate words from the answer list.

Wordcloud Generator

We offer an option to generate a wordcloud for every question asked, which can be downloaded later.

Don't want a wordcloud?

We also offer the option to download a list of all the answers separately.

Character Limit

The host can specify the number of characters he wants to allow to be written in the answer box.

Shareable Links

Form the question, generate a shareable link and share it with people you wanna ask questions from.

Restricted Words

The host the can restrict users from writing certain words apart from the words the website is already restricting.
